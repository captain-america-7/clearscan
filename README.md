# ClearScan (frontend)ho

Minimal, modern PDF viewer using:

- React
- `react-pdf` + `pdfjs-dist`
- Tailwind CSS
- `lucide-react`

## Run locally

```bash
npm install
npm run dev
```

## What’s included

- Loads `public/sample.pdf` by default
- Upload your own PDF (`.pdf`) to render it client-side
- Mocked “API” latency + occasional error to exercise loading/error UI
